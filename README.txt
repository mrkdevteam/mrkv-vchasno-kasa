Плагін інтеграції WooCommerce з Kasa.vchasno.com.ua, сервісом програмної реєстрації розрахункових операцій (пРРО).

== Description ==

Плагін дозволяє створювати чеки з замовлень WooCommerce за допомогою сервісу Вчасно.Каса. Зміна відкривається як автоматично так і вручну. Чек створюється автоматично та в ручному режимі. Ведеться логування всіх запитів плагіна.


Для роботи плагіна необхідно мати зареєстрований акаунт на сервісі [Вчасно.Каса](https://kasa.vchasno.com.ua/).


== Інструкція з налаштування ==

1. Встановіть і активуйте плагін.

2. На сторінці налаштувань плагіна обовз'язково вкажіть загальні налаштування:
- токен каси
- прізвище та ім'я касира
- код податку

3. В налаштуваннях правил формування чеків активуйте автоматичне створення чеків для потрібних способів оплати. Обовʼязково вкажіть тип оплати та статус замовлення, при якому створювати чек автоматично. Інакше чек можна буде створити тільки ручну на сторінці замовлення.

4. Натисність "Зберегти".


== Ручне створення чеків ==

1. На сторінці замовлення, зверху в правій боковій панелі у випадаючому меню, виберіть опцію "Створити чек Вчасно.Каса" та натисніть ">"

2. В примітках до замовлення з'явиться повідомлення про створення чека або про помилку.

3. ID чека відображатиметься у відповідному стовпчику на сторінці WooCommerce -> Замовлення, а також додається до Order Custom Fields vchasno_kasa_receipt_url та vchasno_kasa_receipt_id.
